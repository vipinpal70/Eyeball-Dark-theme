# Eyeball Dark

A clean, eye-friendly dark theme designed for long coding sessions.

## Features

- Soft contrast for reduced eye strain
- Clean syntax highlighting
- Minimal and distraction-free

## Installation

Search for **Eyeball Dark** in VS Code Extensions.
